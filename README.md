# 实现效果

<a href="https://lzxjack.gitee.io/todolist/" target="_blank">[→ 点击此处预览效果 ←]</a>

- 在上方输入栏中输入待办事项，按`回车键`即可添加到`正在进行`中
- `正在进行`中点击每项前面的方格，即可标记为`已经完成`
- `已经完成`中点击每项前面的方格，重新回到`正在进行`
- 点击每项最右边的`圆圈`，删除事项
- 基于`localStorage`本地存储，关闭浏览器再打开，数据不会丢失
